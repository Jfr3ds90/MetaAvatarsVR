TLD LOD Generator v1.0.0
=======================

Professional LOD (Level of Detail) generation tool for Unity.

INSTALLATION:
1. Import the package
2. Access via Tools > TLD Tools > LOD Generator
3. See Documentation folder for complete guide

QUICK START:
1. Select a GameObject with meshes
2. Click "Generate LODs"
3. Done! Your object now has optimized LODs
